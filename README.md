Campus Event Discovery & Participation Tracker

A web-based platform that helps students discover, register, and track campus events such as workshops, hackathons, and seminars in one centralized system. This is a team project developed as part of a Design Thinking initiative.

Features
Event search and filtering
Bookmark / save events (localStorage)
Event details with registration flow
QR-based attendance simulation
Dashboard with participation tracking
Tech Stack
HTML, CSS, JavaScript
LocalStorage for data persistence

Goal
To solve the problem of scattered event information by providing a centralized and user-friendly platform for campus engagement.

Status
Prototype / Frontend implementation for Design Thinking project
